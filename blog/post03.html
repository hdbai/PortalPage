Typeface Blog Post

April 25th, 2017

Roboto belongs to Google font family and the designer is Christian Robertson. The reason it is developed is because Google needs a system font for Android system. From the appearance of Roboto font, we can see that the text is carefully adjusted for mobile view point. Google uses this font on its Nexus5 website here From Nexus5 website, it is very easy to find the spec you want for this product. In order to create a website clean and simple, roboto font is required. I really like Roboto font because its style is much simple. Also, Google has a lot of website using this font, so roboto is getting more and more popular. As we can see, this type of font is getting used by a lot if tech companies. San-serif property makes it very helpful on advertising industry. Most important, people can use roboto font for free.
