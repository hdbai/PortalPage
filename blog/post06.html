Responsive Layout Post

May 24th, 2017

PACK is a website that provides tons of images about dogs. When the screen width > 1280 pixels, it has 4 dog images per row. When the screen width > 960 pixels, it has 3 dog images per row. When the screen width > 640 pixels, it has 2 dog images per row. When the screen width <640 pixels, it has 1 dog images per row, and its navigation bar on the top of the page changes to a hamberg menu.

PACK set their margin and padding to 0 for their picture boxes so that they can maximize the number of pictures they can show on the client's screen. Also, the text is changing while the width of screen is changing. As you can see from the sceenshotsm The "Meet Pack" is getting larger while the screen's width is getting small.
