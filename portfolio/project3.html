Photobooth

Tool: HTML/CSS/NodeJS

Problem:

Webpage in which the user uploads photos to a database. The user can add and delete labels for the photos by hand. Each image uploaded will get several labels from Google Cloud Vision API.

Solution:

The solution is to create many different XHLHttpRequests for uploading image, adding label and deleting label. Looking the documents for Google Cloud Vision API can be very helpful.
