Weather App

Tool: HTML/CSS/NodeJS

Problem:

A website uses the Yahoo weather API to make a weather widget. By default, we see the 5-day weather forecast. Pushing the arrow on the right moves all the days one over to the left, so that today disappears and we see the five days following today. A left arrow should appear to let the user shift back, as well.

Solution:

The solution to this problem is creating a JavaScript function for accessing the Yahoo weather API and changing the DOM value accordingly.
